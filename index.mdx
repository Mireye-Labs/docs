Mireye Earth gives you provenance-tagged geospatial data for any US coordinate — elevation, flood risk, wildfire indicators, buildings, utilities, and more — sourced exclusively from authoritative federal datasets. Every field carries a citation you can audit.

<Steps>
  <Step title="Call the API">
    Hit `https://mireye-earth.fly.dev/v1/ask` with a coordinate and a question. No API key needed in V1.
  </Step>
  <Step title="Read the citations">
    Every answer includes a `citations` array — source agency, source URL, fetch timestamp, and confidence.
  </Step>
  <Step title="Fetch raw fields">
    Use `/v1/fetch` with named fields or a preset to get structured values with per-field provenance.
  </Step>
  <Step title="Connect your agent">
    Install the MCP server to give Claude Desktop or Cursor native access to both tools.
  </Step>
</Steps>
